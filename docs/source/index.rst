Construire un environnement de dev sur mesure avec Python et Docker
===================================================================

Cette documentation a pour objectif d'enseigner aux étudiants en développement
web python et en data science comment utiliser Docker dans leur contexte professionnel. 
En effet, Docker apporte au développeur une grande flexiblé lui permettant de mettre 
en place des environnements de projet sur mesure. Dans un contexte d'apprentissage, cette
technologie permet aux apprenants d'expérimenter en local avec des architectures 
techniques proches des environnements qu'ils rencontreront en production. 

Ces pages ont pour objectif d'illustrer différents aspects de l'utilisation de
Docker, depuis l'installation de l'outils jusqu'à la création et au partage de ses propres
conteneurs. La stratégie utilisée est d'emmener le lecteur à travers un série d'exemples
pratiques permettants d'illustrer différents besoins liés au développement web en python
ou au déploiement de modèles d'apprentissage statistique.
