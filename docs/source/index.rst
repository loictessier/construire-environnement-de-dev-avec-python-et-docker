Construire un environnement de dev sur mesure avec Python et Docker
===================================================================

Cette documentation a pour objectif d'enseigner aux étudiants en développement
web python et en data science à utiliser Docker pour mettre en place des environnements
de projet sur mesure lui permettant d'expérimenter en local avec des architectures 
techniques proches des environnements qu'il rencontrera en production.

Ces pages ont pour objectif d'illustrer différents aspects de l'utilisation de
Docker, depuis l'installation de l'outils jusqu'à la création et au partage de ses propres
conteneurs. La stratégie utilisée est d'emmener le lecteur à travers un série d'exemples
pratiques permettants d'illustrer différents besoins du développement pour le web
en python.
